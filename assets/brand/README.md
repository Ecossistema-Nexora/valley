# Valley Brand Assets

## Logomarca Oficial

Arquivo canonico:

- `assets/brand/logo-valley-official.png`

Espelho usado pelo cockpit web:

- `admin/assets/logo-valley-official.png`

## Uso

- Usar a logomarca oficial em contextos institucionais, cockpit, splash screen, app icon source e materiais de release.
- Preservar proporcao e nao recortar a montanha, estrela ou assinatura Valley.
- Em interfaces pequenas, usar a marca como bloco visual quadrado com raio controlado.
- Em fundos claros, manter uma area escura ao redor da marca quando a leitura perder contraste.

## Paleta Base

- Night: `#07051F`
- Cosmic: `#151047`
- Violet: `#6F2CFF`
- Lilac: `#BB8CFF`
- Cyan: `#20C8F3`
- Snow: `#FFFFFF`

## Direcao Visual

A identidade visual do Valley deve parecer:

- premium
- tecnologica
- noturna
- confiavel
- modular

A interface nao deve virar apenas um gradiente roxo. A marca usa contraste entre noite, neve, violeta e ciano, com superficies legiveis e sinais de status claros.
